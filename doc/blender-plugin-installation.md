### Installing the OpenRCT2-VehicleGenerator Blender Plugin

1. Download the latest version of plugin [here](https://github.com/alex-parisi/OpenRCT2-VehicleGenerator/releases/latest) for your respective platform (Windows/MacOS/Linux).
2. Open Blender
3. Go to File --> Preferences:

   <img src="_static/install1.png" width="300">
4. Click the downwards pointing arrow in the top-right, and select "Install from Disk":

   <img src="_static/install2.png" width="500">
5. Select the `.zip` file you downloaded in Step 1.
